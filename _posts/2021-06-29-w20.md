<pre>
1. 실수형 상수
: 실수는 오차가 있다
: +104.2, -3, 20.4
: 소수점 표기, 지수 표기

2. 문자 상수
: 'A'

3. 문자열 상수
: "APPLE"

4. 정수
: 2(0xb) bin , 8(0xo) oct, 10(int), 16진법(0xh) hex
: 정수->2진수 (4 byte = 32 bit)
: 10 -> 0000\1010
        MSB \ LSB
: -10 -> 2의 보수로 저장
        11110101 + 1
      -> 1111\0110
